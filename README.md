# TodoAppV2

A modern, robust Todo management application built with Kotlin and Jetpack Compose. This project demonstrates a clean, scalable Android architecture using current industry best practices.

## 🚀 Overview
TodoAppV2 helps users organize subjects and tasks efficiently. It emphasizes reactive UI patterns, offline-first data handling, and clear separation of concerns between data, domain, and presentation layers.

## 🛠️ Tech Stack & Architecture
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Declarative UI)
- **Dependency Injection:** Hilt 
- **Local Database:** Room (Offline-first approach)
- **Architecture:** MVVM with a single-source-of-truth Repository pattern
- **Async Programming:** Kotlin Coroutines & Flow
- **Navigation:** Jetpack Compose Navigation

## 🧪 Testing
- ViewModel unit tests using `kotlinx-coroutines-test`
- Fake repositories for deterministic state testing
- StateFlow-based UI state verification
